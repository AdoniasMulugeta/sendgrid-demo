### A Simple Sendgrid Integration Demo
#### Tech used
- Node.js
- Express.js
- Mocha for testing
- Chai for assertion
- Sinon for stubbing
- Dotenv

#### Code organization
- ES6 Classes are used to organize code
- Factory patter is used for ease of test and to demonstrate modular implementation
